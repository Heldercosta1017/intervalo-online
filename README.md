# Calculadora de Intervalos A → E

Esta é uma calculadora online que permite calcular os valores intermediários (B, C, D...) entre dois números (A e E).

## Funcionalidades
- Escolher A e E
- Definir número de divisões
- Escolher método de arredondamento: Decimal, Inteiro ou Bruto
- Resultado exibido na página

## Como usar
1. Abrir `index.html` no navegador
2. Inserir os valores de A e E
3. Definir número de divisões
4. Selecionar método de arredondamento
5. Clicar em "Calcular Intervalos"

## Hospedagem online
Para colocar online, basta subir o repositório no GitHub e habilitar **GitHub Pages** em **Settings → Pages**.
